<task type="checkpoint:human-action">
  <name>Task 2: Configure NPM_TOKEN secret</name>
  <files></files>
  <action>
User needs to add NPM_TOKEN secret to GitHub repository:

1. Go to https://www.npmjs.com/settings/[username]/tokens
2. Generate new "Automation" token
3. Go to GitHub repo → Settings → Secrets and variables → Actions
4. Add secret named `NPM_TOKEN` with the token value

For scoped packages (@org/name), ensure npm account has publish access to the scope.
  </action>
  <verify>echo "Verify in GitHub repo settings"</verify>
  <done>NPM_TOKEN secret configured in GitHub</done>
</task>
